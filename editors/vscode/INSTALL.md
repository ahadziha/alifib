# alifib syntax highlighting for VS Code

## Install

```
ln -s "$(cd "$(dirname "$0")" && pwd)" ~/.vscode/extensions/alifib
```

Or manually:

```
ln -s /path/to/editors/vscode ~/.vscode/extensions/alifib
```

Then reload VS Code (Cmd+Shift+P → "Reload Window").

## Uninstall

```
rm ~/.vscode/extensions/alifib
```
